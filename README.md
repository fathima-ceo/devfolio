# Fathima H — Portfolio

This is a responsive personal portfolio built with HTML, CSS and JavaScript. It includes sections for Home, About, Skills, Internships, Projects, Education, Certificates, Resume and Contact.

Quick start:

```bash
# Open the site in your browser (Windows)
start index.html
```

Deploy / Add to GitHub
----------------------

To initialize git, commit, and push this folder to your GitHub repository at `https://github.com/fathima-ceo/devfolio`, run:

```bash
git init
git add .
git commit -m "Initial portofolio site"
git branch -M main
git remote add origin https://github.com/fathima-ceo/devfolio.git
git push -u origin main
```

What I added:
- Modern hero with visual accent and CTA buttons
- Theme toggle (light/dark) persisted to localStorage
- Animated reveal on scroll for project cards
- Resume preview modal and download

Replace the placeholder `assets/Fathima_Resume.pdf` with your actual resume PDF to enable a full preview and download.
